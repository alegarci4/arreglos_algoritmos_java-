# Arreglos y Algoritmos en Java 

Este repositorio contiene el desarrollo de la tarea dividida en dos partes:

## Parte 1 – Ingeniería Inversa

Se realizó ingeniería inversa al archivo `.jar` utilizando la herramienta JD-GUI.


La documentación y capturas del proceso se encuentran en la carpeta `docs`.



## Parte 2 – Ejercicio Algorítmico

Se desarrolló un algoritmo que permite encontrar:

- El segundo número mayor
- El segundo número menor

Cumpliendo con lo que se pidió:

- Solo una pasada al arreglo
- Sin ordenar el arreglo

Incluye pseudocódigo, explicación y análisis de complejidad.
